# 时态语法

## 动词的各种变形

| 动词的各种形态 | 变形  | 是否可做谓语 |
| -------------- | ----- | ------------ |
| 原型           | do    | 可以         |
| 三单           | does  | 可以         |
| 过去式         | did   | 可以         |
| 现在分词       | doing | 不可以       |
| 过去分词       | done  | 不可以       |
| 不定式         | to do | 不可以       |

## 时态语法

|          | 一般     | 进行时          | 完成时          | 完成进行时          |
| -------- | -------- | --------------- | --------------- | ------------------- |
|          | 做       | 正在做          | 已经做完        | 一直做              |
| 现在     | do/does  | am/is/are doing | have/has done   | have/has been doing |
| 过去     | did      | was/were doing  | had done        |                     |
| 将来     | will do  | will be doing   | will have done  |                     |
| 过去将来 | would do | would be doing  | would have done |                     |
